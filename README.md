## 📌 Overview  
FoodCart is a modern food ordering web application built using React, designed to simulate a real-world food ordering experience with a clean UI and smooth cart interactions.  
It focuses on usability, fast performance, and responsive design for seamless ordering across devices.

<p align="center">
  <img src="https://github.com/aliiashraf777/1red-food-cart-app/blob/main/src/assets/20pr20preview.png" />
</p>

## 🎯 Problem  
- ❌ Complex and cluttered food ordering interfaces  
- ❌ Poor cart interaction experience  
- ❌ Slow UI responsiveness  
- ❌ Weak mobile usability  

## 💡 Solution  
- ✅ Clean and intuitive UI  
- ✅ Smooth cart interaction system  
- ✅ Fast and responsive design  
- ✅ Scalable component architecture  

## ⚡ Key Features  
- 🍔 Food menu listing  
- 🛒 Add/remove cart functionality  
- 💰 Cart total calculation  
- 📱 Fully responsive layout  
- ⚡ Fast UI performance  

## 🛠 Tech Stack  
- ⚛️ React.js  
- ⚡ Vite  
- 🧠 JavaScript (ES6+)  

## 📈 Results / Impact  
- 🚀 Improved ordering experience  
- ⚡ Faster UI interactions  
- 📱 Smooth mobile usability  
- 🎯 Clean and scalable structure
